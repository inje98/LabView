# LabView

### 단축키
- 새 VI 생성 : Ctrl + N 
- VI 좌우 정렬 : Ctrl + T
- 함수 설명 : Ctrl + H
- 도구 팔레트 : Shift + 우클
- 주석 : 빈 공간 더블클릭

### 용어
- 프런트패널 : Ctrl+T 정렬 기준 좌측
- 블록다이어그램 : Ctrl+T 정렬 기준 우측 
- 컨트롤 : 프런트패널에서 블록다이어그램으로 값을 전달할 때 사용하는 입력 변수
- 인디케이터 : 결과값을 사용자에게 보여줄때
- 터미널 : 프런트패널의 컨트롤 값을 블록다이어그램으로 전달하는 기능 수행
